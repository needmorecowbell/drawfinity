# Keyboard Shortcuts

## Tools

| Shortcut | Action |
|----------|--------|
| <kbd>B</kbd> | Brush tool |
| <kbd>E</kbd> | Eraser tool |
| <kbd>R</kbd> | Rectangle shape tool |
| <kbd>O</kbd> | Ellipse shape tool |
| <kbd>P</kbd> | Polygon shape tool |
| <kbd>S</kbd> | Star shape tool |
| <kbd>G</kbd> | Toggle pan/zoom tool |

## Brush Presets

| Shortcut | Action |
|----------|--------|
| <kbd>1</kbd> | Pen |
| <kbd>2</kbd> | Pencil |
| <kbd>3</kbd> | Marker |
| <kbd>4</kbd> | Highlighter |
| <kbd>[</kbd> | Decrease brush size |
| <kbd>]</kbd> | Increase brush size |

## Canvas Navigation

| Shortcut | Action |
|----------|--------|
| <kbd>Space</kbd> + drag | Pan |
| Middle mouse drag | Pan |
| Scroll wheel | Zoom (discrete steps) |
| Trackpad pinch | Zoom (continuous) |
| <kbd>Ctrl</kbd>+<kbd>=</kbd> | Animated zoom in |
| <kbd>Ctrl</kbd>+<kbd>-</kbd> | Animated zoom out |
| <kbd>Ctrl</kbd>+<kbd>0</kbd> | Reset zoom to 100% |

## Editing

| Shortcut | Action |
|----------|--------|
| <kbd>Ctrl</kbd>+<kbd>Z</kbd> | Undo |
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>Z</kbd> | Redo |

## Panels & UI

| Shortcut | Action |
|----------|--------|
| <kbd>Ctrl</kbd>+<kbd>K</kbd> | Toggle connection panel |
| <kbd>Ctrl</kbd>+<kbd>B</kbd> | Toggle bookmark panel |
| <kbd>Ctrl</kbd>+<kbd>D</kbd> | Quick-add bookmark |
| <kbd>Ctrl</kbd>+<kbd>,</kbd> | Toggle settings panel |
| <kbd>Ctrl</kbd>+<kbd>`</kbd> | Toggle turtle graphics panel |
| <kbd>F3</kbd> | Toggle FPS counter |

## Export & Navigation

| Shortcut | Action |
|----------|--------|
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>E</kbd> | Export PNG |
| <kbd>Ctrl</kbd>+<kbd>W</kbd> | Return to home screen |
| <kbd>Escape</kbd> | Return to home screen |

## Display

| Shortcut | Action |
|----------|--------|
| <kbd>Ctrl</kbd>+<kbd>'</kbd> | Toggle dot grid |
